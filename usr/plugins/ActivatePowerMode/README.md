### 打字特效插件`ActivatePowerMode`

![演试图片](play.gif)

#### 使用方法

- 下载本插件，解压放到`usr/plugins/`目录中
- 文件夹名改为`ActivatePowerMode`
- 登录管理后台，激活插件

#### 插件说明
插件基于`activate-power-mode.js`
